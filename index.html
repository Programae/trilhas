Trilhas Programaê!

Seja bem vindo! Aqui você encontrará os "rascunhos" das propostas trilhas que estamos desenvolvendo para o novo site do Programaê! Se você ainda não sabe o que é Programaê! recomendo começar por este link.

Cada uma das apresentações abaixo aprofunda em um tema específico. Em cada um destes temas propomos ao usuário uma missão a ser desenvolvida, por exemplo, a trilha WEB #1 convida o usuário a "contruir um site". O mesmo modelo aplica-se para os outros temas, por meio da missão inspiramos o usuário a criar algo.

Atualmente definimos os temas a partir de três frentes amplas (web, mobile e jogos) e duas específicas (professores e infantil). O conteúdo associado a cada uma das missões pode ser o mais variado possível, tais como: artigos, vídeos no youtube, podcasts, plataformas de exercícios online, MOOCs, etc; O importante neste caso é que os conteúdos façam sentido dentro do escopo apresentado.

Neste momento estamos buscando conteúdos para cada um destes temas, lembrando que para ser elegível a recomendação precisa ser:

  • Gratuito
  • Traduzido para a língua Portuguesa (dependendo da relevância podemos considerar um esforço para tradução)
